SIMBAD and VIZIER Web Scraper 
This script makes a connection with then Simbad and Vizier databases. It requests the following data
(HR identifier, HD identifier, RA and DEC coordinates, B magnitude, V magnitude, spectral type, b-y color index,
m1 color index, c1 color index and Hbeta index) and outputs an Excel file with the
information requested (optional)
